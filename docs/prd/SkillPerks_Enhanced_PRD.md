# SkillPerks — Enhanced Product Requirements Document
## Affiliate Learning Platform for Developers & Tech Professionals

---

## 1. EXECUTIVE SUMMARY

**Product Name:** SkillPerks  
**Domain:** skillperks.org  
**Tagline:** Smarter Savings on Tech Skills and Career Growth

**Mission:** Empower developers, data scientists, and tech learners to make informed decisions about online learning investments through transparent comparisons, verified coupons, and expert insights.

**Primary Goals:**
1. Drive affiliate conversions for learning platforms
2. Build topical authority in tech education and career development
3. Provide genuine value through course previews and comparison tools
4. Maintain user trust through transparency and FTC compliance

## 2. TARGET AUDIENCE

### Primary Segments
1. **Junior Developers (0-2 years)**
   - Seeking interview prep and foundational skills
   - Budget-conscious, ROI-focused
   - Looking for structured learning paths

   - Upskilling in system design, cloud, AI/ML
   - Career advancement focused

3. **Career Switchers**
   - Transitioning to tech from other fields
   - Need comprehensive, beginner-friendly content

   - Preparing for technical interviews

   - Primarily India, but global reach
   - Price-sensitive markets

## 13. DATA STRUCTURE

### Course Platform Schema
```typescript
interface Platform {
  id: string;
  name: string;
  slug: string;
  logo: string;
  rating: number;
  reviewCount: number;
  description: string;
  pricing: {
    monthly: number;
    annual: number;
    lifetime?: number;
    freeTier: boolean;
  };
  features: string[];
  courseCount: number;
  categories: string[];
  certificationType: string;
  freeTrial: {
    available: boolean;
    duration: number;
  };
  refundPolicy: string;
  affiliateLink: string;
  lastUpdated: Date;
}
```

### Coupon Schema
```typescript
interface Coupon {
  id: string;
  platformId: string;
  code: string;
  discount: {
    type: 'percentage' | 'fixed';
    value: number;
  };
  description: string;
  expiryDate: Date;
  isExclusive: boolean;
  successRate: number;
  timesUsed: number;
  restrictions: string[];
  affiliateLink: string;
}
```

### Review Schema
```typescript
interface Review {
  platformId: string;
  overallRating: number;
  ratings: {
    courseQuality: number;
    platformUsability: number;
    valueForMoney: number;
    support: number;
    certification: number;
  };
  pros: string[];
  cons: string[];
  bestFor: string[];
  notRecommendedFor: string[];
  author: string;
  publishDate: Date;
  lastUpdated: Date;
}
```

## 14. AFFILIATE COMPLIANCE

### FTC Guidelines
1. **Clear Disclosure**
   - Visible on every monetized page
   - Above the fold
   - Plain language
   - No buried disclaimers

2. **Sample Disclosure:**
   > "SkillPerks partners with online learning platforms and may earn a commission when you purchase through our links. This helps us keep the site free and doesn't affect your price. We only recommend platforms we've personally tested and believe provide value."

3. **Placement:**
   - Header notification bar
   - Before first affiliate link
   - Footer
   - Dedicated disclosure page

### Ethical Guidelines
- Never recommend solely for commission
- Test platforms before reviewing
- Disclose sponsored content
- Update reviews regularly
- Allow user comments/feedback
- Respond to complaints
